# 🎭 Mad Libs Horror Generator

An interactive web application that starts as a fun Mad Libs game and gradually transforms into a psychological horror experience.

Built with **Angular (frontend)** and **Spring Boot (backend)**.

---

## 🚀 Features

### 🎮 Core Gameplay
- Generate funny stories using user-provided words
- Random word generator option
- Score tracking system
- Typewriter-style text animation

### 🎨 UI/UX
- Modern responsive UI
- Smooth transitions and animations
- Dynamic state changes (loading, typing, horror mode)

### 🔊 Sound Design
- Calm background music
- Button click sounds
- Horror sound transitions
- Silence-based tension before jumpscare
- Final scream effect

### 👁️ Horror System
- Hidden easter egg trigger
- Progressive UI transformation
- Warning popup before final sequence
- Screen fade to black effect
- Jumpscare animation
- Glitching ending text

### 🔁 Game Flow
1. Normal gameplay
2. Hidden horror trigger
3. Warning phase
4. Silence (tension)
5. Final action (one last story)
6. Jumpscare
7. Ending screen
8. Click to restart
